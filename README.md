## Описание программы

Random Task Generator - это настольное приложение с графическим интерфейсом, разработанное на языке Python с использованием библиотеки Tkinter.

Программа предназначена для:

- генерации случайных задач;
- распределения задач по категориям;
- добавления пользовательских задач;
- удаления задач;
- хранения истории сгенерированных задач;
- сохранения и загрузки данных через формат JSON.

## Основной функционал программы

### 1. Генерация случайных задач
Пользователь может выбрать категорию:
- Учёба
- Спорт
- Работа
- Все категории
После нажатия кнопки программа случайным образом выбирает задачу из выбранного списка.
### 2. Добавление новых задач
Пользователь может:
- Ввести собственную задачу
- Выбрать категорию
- Добавить её в программу
### Проверка:
- Пустые строки запрещены
- Дубликаты задач не добавляются
### 3. Удаление задач
Можно удалить любую задачу из общего списка.
### 4. История задач
Каждая сгенерированная задача сохраняется в историю.

## Примеры использования программы

### Пример 1: Генерация задачи
1. Выберите категорию Спорт
2. Нажмите кнопку Сгенерировать задачу
3. Программа выдаст:

 [Спорт] Сделать зарядку

---

### Пример 2: Добавление новой задачи
Пользователь вводит:

 Пробежать 5 км

Выбирает категорию:

 Спорт

После добавления задача появляется в списке.

---

### Пример 3: Удаление задачи
1. Выберите задачу из списка
2. Нажмите кнопку Удалить выбранную задачу

---

### Пример 4: Сохранение
После закрытия программы данные автоматически сохраняются в JSON-файлы.
